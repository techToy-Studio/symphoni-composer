# Composer Interface

## Main Components
- **Timeline**: Where you place and arrange notes
- **Toolbar**: Contains editing tools and options
- **Preview Space**: See how your level will look
- **Properties Panel**: Adjust note and level settings

## Timeline Features
- Note placement grid
- Time markers
- Section markers
- Playback controls

## Toolbar Tools
- Note placement tools
- Editing tools
- View controls
- Playback controls

## Properties Panel
- Note properties
- Level settings
- Environment settings
- Audio settings
